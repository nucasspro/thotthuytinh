Hướng dẫn cài đặt
1) Mở Project bằng Visual Studio
2) Vào Tool -> Nuget Package Manager -> Package Manager Console
3) Chạy lệnh cài: Install-Package FluentWPF

Cấu hình Project
1) Thêm dòng sau vào file xaml
xmlns:fw="clr-namespace:SourceChord.FluentWPF;assembly=FluentWPF"

2) Thêm ResourceDictionary vào file App.xaml
<ResourceDictionary>
     <ResourceDictionary.MergedDictionaries>
          <ResourceDictionary Source="pack://application:,,,/FluentWPF;component/Styles/Controls.xaml" />
     </ResourceDictionary.MergedDictionaries>
</ResourceDictionary>

3) Sửa file xaml
<Window x:Class="GetOrderApp.MainWindow" 
thành 
<fw:AcrylicWindow x:Class="GetOrderApp.MainWindow"

4) Sửa file xaml.cs
class MainWindow : Window 
thành 
class MainWindow
